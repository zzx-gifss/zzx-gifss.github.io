空间复杂度

算法在运行过程中临时占用存储空间大小的量度

栈

后进先出的数据结构

js中没有栈，可以用array实现栈功能

什么场景下使用栈

  1. 需要后进先出的场景

比如 十进制转二进制 判断字符串括号是否有效 函数调用堆栈

  • 十进制转二进制

后出来的余数反而排到前面

把余数依次入栈，依次出栈

  • 有效的括号

越靠后的左括号，对应的有括号越靠前

遇到左括号入栈，遇到右括号出栈，最后栈空了就是合法的

  • 函数调用堆栈

最后被调用的函数，最先被执行。

队列

先进先出

在javascript中没有队列，使用数组实现

队列使用场景

需要先进先出的场景

  • js异步中的任务队列

  • 计算最近请求次数

链表

  • 多个元素组成的列表
  • 元素存储不连续，用next指针连在一起

数组与链表的区别

数组：增删非首尾元素时往往要移动元素

链表：增删非首尾元素，不需要移动元素，只需要更改指针的指向即可

js中没有链表数据结构，使用Object来模拟链表

反转两个节点：将n+1的next指向n 反转多个节点：双指针遍历链表，重复上述操作

反转链表

双指针一前一后遍历链表，反转双指针

环形链表

定义快指针和慢指针，快指针一次走两步，慢指针一次走一步，如果有环他们会相逢

集合

一种无序且唯一的数据结构 set（ES6）

常用操作

  • 去重
  • 判断某元素是否存在
  • 求交集

前端中的集合

使用set的方法 new,add,delete,has,size

迭代set：set和array互转

const myarray=[...mySet]

const myarray=Array.from

树

js中没有树，但是可以用object和array构建树

树的常用操作：深度/广度优先遍历 先中后序遍历

深度优先遍历：尽可能深的搜索树的分支

  • 访问根节点
  • 对根节点的children挨个深度遍历

广度优先遍历：先访问离根节点最近的节点

  • 新建一个队列，把根节点入队
  • 把队头出队并访问
  • 把队头的children挨个入队
  • 重复以上步骤，直到队列为空

二叉树的先中后序排列

二叉树：树中每个节点最多只能有两个字节点

在js中使用Object模拟二叉树

先序遍历：

  • 访问根节点
  • 对根节点的左子树先序遍历
  • 对根节点右子树进行先序遍历

1 -> 2 ->3 ->4 -> 5 ->6 ->7

中序遍历

  • 对根节点的左子树进行中序遍历
  • 访问根节点
  • 对根节点的右子树进行中序遍历

1 -> 2 ->3 ->4 ->5 ->6 ->7

后序遍历

  • 对根节点的左子树进行后序遍历
  • 对根节点的右子树进行后序遍历
  • 访问根节点

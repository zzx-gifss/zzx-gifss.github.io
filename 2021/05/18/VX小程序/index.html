pages/index

index.js index.json index.wxml index.wxss

四种文件

wxml 显示页面骨架

wxss 页面样式

js 动态页面

json 进行页面配置 可以配置颜色相关的，可以配置很多东西

四种文件不是必须的，例如没有配置或者没有样式就可以删除相应文件

在小程序里不用引入样式，保证四个文件是同名的，在wxml中就可以直接使用wxss，不用导入

app.js app.json app.wxss

这三个文件的名字不能修改 这三个是程序级别的文件

上面四个是页面级别的文件，在app.json中的配置在所有页面都会生效

在程序级别修改文件样式以后，在页面级单独修改时，覆盖程序级别样式文件，就近原则，离页面越近配置生效。

页面配置覆盖全局配置。

具体流程

  1. post-detile xml 给图片添加点击事件
  2. 在js添加点击事件 假设当前状态是未收藏，用户点击变为收藏状态
  3. 将用户收藏的状态保存在缓存中 我们要设定收藏的是哪篇文章
  4. 由于我们有多篇文章，我们需要设计一个数据结构，可以表示多篇文章是否被收藏，在缓存中设置js对象，设置key为文章id,设置flag定义到底是否收藏
  5. key可以在onload函数中获取，但是没法和点击的函数共享一个数据，因为是两个不同的函数。可以在data中进行中转，在data中设置变量（动态访问属性)
